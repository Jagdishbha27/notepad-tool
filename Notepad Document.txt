# Online Notepad Tool

![Notepad Tool](screenshots/notepad-tool.png)

## Introduction

Online Notepad is a web-based text editor that runs entirely in your browser. It allows you to create, edit, format, and save notes, documents, or code snippets. Whether you need a simple notepad for quickly jotting down ideas or a more advanced text editor for creating formatted documents, Online Notepad has you covered.

## Features

### Editing Features
- **Rich Text Editing**: Bold, italic, underline, strikethrough, and other formatting options
- **Text Alignment**: Left, center, right, and justify alignment options
- **Font Customization**: Various font, size, and color options
- **Theme Options**: Light, dark, blue, green, sepia, and other themes
- **Line Spacing**: Single, 1.5, double, and other line spacing options

### File Operations
- **New Document**: Quickly create a new document (Ctrl+N)
- **Open File**: Open existing text files (Ctrl+O)
- **Save**: Save your work (Ctrl+S)
- **Print**: Print your document (Ctrl+P)
- **Download**: Download your document in text (.txt) format

### Search and Replace
- **Find and Replace**: Search and replace text throughout the document
- **Case Sensitive Search**: Search with match case option
- **Word Search**: Search for whole words
- **Previous/Next**: Navigate between search results

### Autosave and Recovery
- **Automatic Saving**: Automatically saves your work every 30 seconds
- **Session Recovery**: Recover your work even after browser closure

### Sharing
- **Document Sharing**: Share your document via link
- **Social Sharing**: Share on Facebook, Twitter, WhatsApp, LinkedIn

### Other Features
- **Dark Mode**: Optimal for low-light environments
- **Keyboard Shortcuts**: For quick and efficient editing
- **Word and Character Count**: Document analysis
- **Mobile Friendly**: Use on any device

## Setup Instructions

### Local Setup

1. **Clone Repository**
   ```bash
   git clone https://github.com/yourusername/online-notepad.git
   cd online-notepad
   ```

2. **Open Files**
   - Open `index.html` in a web browser (preferably Google Chrome, Firefox, or Edge)

### Using a Web Server

1. **Simple Web Server**
   ```bash
   # With Python 3
   python -m http.server 8000
   
   # Or with Node.js
   npx serve
   ```

2. **Access in Web Browser**
   - Navigate to `http://localhost:8000` in your browser

### Deploying on Hosting

1. **Upload Files**
   - Upload all files (HTML, CSS, and JS) to your hosting provider
   - Follow the instructions for installation

## How to Use

### Basic Editing

1. **Creating a New Document**
   - Click on the "New Document" button or use Ctrl+N
   - Start typing in the editor area

2. **Text Formatting**
   - Select text and use formatting options from the toolbar
   - Bold (Ctrl+B), Italic (Ctrl+I), Underline (Ctrl+U)

3. **Changing Font and Theme**
   - Select font family, size, and color according to your preference
   - Try different themes to change the editor's appearance

### File Operations

1. **Saving a Document**
   - Click on the Save button or press Ctrl+S
   - Enter a name for your note

2. **Opening a File**
   - Click on the Open File button or press Ctrl+O
   - Select a file from your device

3. **Downloading a File**
   - Click on the Download Document button (Ctrl+D)
   - The file will be downloaded to your computer in .txt format

4. **Printing a Document**
   - Click on the Print button or press Ctrl+P
   - View the print preview and click on the Print button

### Search and Replace

1. **Finding Text**
   - Click on the Find & Replace button or press Ctrl+F
   - Enter text in the search box
   - Customize search options (match case, whole word)
   - Use Find Next (F3) and Find Previous (Shift+F3) for navigation

2. **Replacing Text**
   - Follow the search steps above
   - Enter replacement text in the Replace box
   - Click on the Replace button to replace the current match (Ctrl+H)
   - Or click on Replace All to replace all matches (Ctrl+Alt+H)

### Notes Management

1. **Saving Notes**
   - In the Saved Notes section, enter a name for your note
   - Click on the Save Note button

2. **Loading Saved Notes**
   - In the Saved Notes section, click on the Load button next to a note

3. **Deleting Notes**
   - Click on the Delete button next to a saved note

### Document Sharing

1. **Sharing via Link**
   - Click on the Share button (Ctrl+Shift+S)
   - Click on the Copy Link button or choose a social media icon
